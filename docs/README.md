## 📊 技术栈

### 后端技术
- **Web框架**: Django 4.2.16

- **AI模型**: RAG + 火山引擎Doubao

- **可视化**: Matplotlib + Seaborn + Plotly

- **数据分析**: Pandas + NumPy

- **向量化**: Sentence Transformers
