## 🎯 项目概述
基于Django框架的房源数据分析系统，支持MySQL和MongoDB双数据库架构，提供完整的数据可视化、分析和预测功能。

## 🏗️ 系统架构

### **双数据库架构**
```
┌─────────────────┬─────────────────┐
│   MySQL版本     │   MongoDB版本   │
│   /app/         │   /mongo/       │
├─────────────────┼─────────────────┤
│ Django ORM      │ MongoEngine     │
│ 关系型数据库    │ 文档型数据库    │
│ 10,783条记录    │ 10,783条文档    │
└─────────────────┴─────────────────┘
```

### **技术栈**
- **后端框架**: Django 4.2.16
- **数据库**: MySQL 8.0 + MongoDB 8.0.5
- **ORM**: Django ORM + MongoEngine
- **前端**: Bootstrap + ECharts + jQuery
- **缓存**: Django LocMemCache
- **爬虫**: Scrapy框架
